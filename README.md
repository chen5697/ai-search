# ai-search
这是一个给公司做的一个搜索项目demo，主要实现了NL2SQL、RAG、问题推荐、和简单的聊天界面，界面包括用户登录、选择知识库、切换模型问答等，所有聊天会话放在MYSQL数据库留档。
